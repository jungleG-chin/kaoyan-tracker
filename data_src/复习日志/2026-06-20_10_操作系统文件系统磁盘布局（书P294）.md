# 操作系统文件系统磁盘布局（书P294）

## K_文件系统磁盘布局 · 408 · 文件系统磁盘布局：MBR、超级块、inode区、数据区、空闲空间管理 [pass · 8/10]

- **章节**：【OS】四、文件管理
- **题目**：文件系统的磁盘布局是什么样的？各区域的数据结构和作用分别是什么？
- **你的回答**：1. 不同区域数据结构不同，inode区数据结构统一 ✓
2. inode靠mode参数区分目录还是普通文件 ✓
3. inode分区前有空闲空间管理，用位图记录哪些块空闲 ✓
4. 主引导记录(MBR)在0号扇区，存放引导程序 ✓
5. 还有一个引导块 ✓
6. inode区后面是根目录，根目录位置固定 ✓
7. 根目录记录文件/目录的映射，方便按目录名找文件 ✓（本质上是目录项映射）
- **标准答案**：【文件系统磁盘布局（以ext2/Unix风格为例）】

整体布局（从低地址到高地址）：
┌──────────┬──────────┬──────────┬──────────┬────────────────┬────────────────┐
│  0号扇区  │  引导块  │ 超级块   │ inode位图│  inode区       │  数据块区      │
│  (MBR)   │ (Boot)  │(Superblk)│+块位图  │  (inode table) │  (data blocks) │
└──────────┴──────────┴──────────┴──────────┴────────────────┴────────────────┘

【各区域详解】

1. 0号扇区 — MBR（主引导记录）
   - 你的理解：✓ 正确
   - 存放：引导程序 + 磁盘分区表
   - 作用：计算机开机后BIOS第一个读这里，引导OS加载
   - 注意：MBR是磁盘级别的，不属于某个分区

2. 引导块（Boot Block）
   - 你的理解：✓ 正确，MBR和引导块是两个不同的东西
   - 位置：每个分区的第一块
   - 作用：分区级引导代码（如果这个分区是可启动分区）
   - 不可启动的分区该块为空，但依然保留位置

3. 超级块（Super Block）
   - 你没有提到这个！——重要遗漏
   - 作用：描述整个文件系统的元数据
     · 文件系统总大小
     · 每块大小
     · inode总数/空闲inode数
     · 数据块总数/空闲块数
     · 文件系统魔数（标识类型）
   - 超级块损坏 = 整个文件系统崩溃，所以通常有备份副本

4. 空闲空间管理区（位图区）
   - 你的理解：基本正确，但要区分两种位图
   - inode位图：记录哪些inode已用/空闲
   - 数据块位图：记录哪些数据块已用/空闲
   - 位图中 0=空闲，1=已用

5. inode区（inode Table）
   - 你的理解：✓ 正确（数据结构统一，都是inode结构）
   - 每个inode固定大小（如128字节），存储文件元数据：
     · mode（类型+权限）：区分普通文件/目录/符号链接等
     · 文件大小
     · 时间戳（创建/修改/访问时间）
     · 数据块指针（直接块/间接块/二级间接块）
   - 注意：inode不存文件名！文件名在目录文件里

6. 数据块区（Data Blocks）
   - 存放文件的实际内容
   - 目录文件的数据块存的是：文件名→inode号 的映射表
   - 普通文件的数据块存的是：文件实际数据

7. 根目录
   - 你的理解：✓ 位置固定（通常是inode号=2）
   - 根目录本质上是一个特殊的目录文件，存放的是:
     [文件名] → [inode号] 的映射
   - 所有文件查找都从根目录的inode开始

【你的理解中需要补充的点】
1. 漏掉了超级块（Super Block）——考试必考！
2. inode区后面不一定紧跟根目录数据，根目录是数据块区里inode=2指向的数据块
   更准确说法：根目录是第2号inode，其数据在数据块区里

【408考试高频考点】
- 超级块存什么（选择题）
- inode不存文件名（判断题，容易错）
- 目录文件的内容=文件名到inode号的映射
- MBR vs 引导块的区别（磁盘级 vs 分区级）
- 两种位图：inode位图 vs 数据块位图
- **错误**：否

## K_拐点判定方法 · 数学二 · 拐点判定 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：拐点判定：嫌疑点有哪些？如何确认一个点是拐点？
- **你的回答**：拐点是凹凸性发生改变的点。嫌疑点：二阶导等于零或二阶导不存在的点。若二阶导函数在这一点左右异号，则该点是拐点。
- **标准答案**：✓ 标准答案：
1. 拐点定义：凹凸性发生改变的点。
2. 嫌疑点：f(x)=0 或 f(x) 不存在的点。
3. 判定：若 f(x) 在该点左右两侧异号，则该点为拐点。
4. 注意：f(x)=0 不一定就是拐点（如 f(x)=x^4 在 x=0 处二阶导为零但非拐点），必须验证左右异号。
- **错误**：否
