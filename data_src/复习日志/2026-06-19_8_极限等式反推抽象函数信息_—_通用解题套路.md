# 极限等式反推抽象函数信息 — 通用解题套路

## K_极限等式反推套路 · 数学二 · 极限等式反推抽象函数信息（Q9、Q13核心套路） [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：已知 lim_{x→0} 分子/分母 = 常数（或0），且分母→0，如何反推抽象函数 f(x) 在 x=0 处的信息？
- **你的回答**：分式极限为常数，分母趋近0 → 分子必须也趋近0，借此反推 f(0)、f撇(0) 等。这是 Q9 和 Q13 的共同套路。
- **标准答案**：【通用解题套路 — 极限等式反推 f 信息】

类型：lim_{x→a} 分子/分母 = c（c 为有限常数），且分母→0

核心原理：分母→0，若极限存在且有限，则分子也必须→0
否则：分子→非零 / 分母→0 = ∞，矛盾

【标准四步流程】

Step 1：由分母→0 + 极限存在有限 ⇒ 分子→0
  → 得到 f(a) 的值（通常是 f(0)=0 或 f(0)=某常数）

Step 2：极限式变形，用等价无穷小替换分母
  → 常见：ln(1+x)~x，e^x-1~x，e^{x^2}-1~x^2，sin x~x

Step 3：用导数定义反推 f撇(0)
  → f撇(0) = lim_{x→0} [f(x)-f(0)]/x
  → 目标：把已知极限式化成这个形式

Step 4：若还有二阶极限，继续推 f两撇(0)
  → f两撇(0) = lim_{x→0} [f撇(x)-f撇(0)]/x
  → 或直接用已知极限式 + 泰勒展开

【例题验证】

→ Q9：lim_{x→0} x^2/f(x) = 1
  Step1：分母 x^2→0，极限=1（有限）
  → f(x) 必须→0（否则 x^2/f(x) 会→0 而不是 1）
  → f(0)=0 ✓，且 f(x)~x^2
  Step3：f撇(0) = lim f(x)/x = lim f(x)/x^2 * x = 1*0 = 0 ✓

→ Q13：lim_{x→0} [f(x)-a]/ln(1+x)=0
  Step1：分母 ln(1+x)~x→0，极限=0
  → 分子 [f(x)-a] 必须是 o(x)（比 x 更高阶无穷小）
  → f(0)=a，且 f撇(0)=0 ✓
  第二个极限：lim [f两撇(x)-1]/(e^{x^2}-1)=2025
  Step1：分母~x^2→0，极限有限 → 分子→0 → f两撇(0)=1 ✓

【速记口诀】
  分式极限存在，分母零则分子零
  等价替换先动手，导数定义最后收
  一阶极限推 f(0)f撇(0)，二阶极限推 f两撇(0)
- **错误**：否
