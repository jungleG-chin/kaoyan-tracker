# 一元函数微分错题（高数研选题 第2章）Q8-Q14

## 高数研选题_第2章_Q8 · 数学二 · 极限函数化简 + 不可导点判断 [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第8题：f(x) = lim_{n→∞} √[n]{1+|x|^n+e^{nx}} 不可导点的个数为？
- **你的回答**：认为 x=0 可导（✓），x=1 不可导（✗），选 B。自己的思路：把 |x|^n 和 e^{nx} 两个式子单独拎出来分类讨论写分段函数；答案思路：用 max 函数表示整个函数，对整个函数进行分段讨论取大头。
- **缺漏**：没有先用 max 化简极限函数就直接猜不可导点。
【驻点定义】：f'(x₀)=0 的点称为驻点。注意：驻点不一定是极值点。
【拐点定义】：曲线凹凸性发生改变的点 (x₀,f(x₀))。判定：f''(x₀)=0 且 f'' 在 x₀ 两侧变号。
- **标准答案**：答案：C. 2 个。
① 先用 n 次根号下 max 性质：f(x)=max{1, |x|, e^x}
② 分段讨论得：f(x)={-x (x≤-1); 1 (-1≤x≤0); e^x (x≥0)}
③ x=-1：左导-1≠右导0 → 不可导
④ x=0：左导0≠右导1 → 不可导
⑤ 共 2 个不可导点。

自己的思路总结：你想到对两个式子分别讨论是对的，但遗漏了'取 max'这个关键操作。答案用 max 函数来表示整个 f(x) 再分段，是更系统的方法。
- **错误**：是

## 高数研选题_第2章_Q9 · 数学二 · 极限等式反推函数性质 + 导数定义 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第9题：已知 f(x) 在 x=0 处连续，且 lim_{x→0} x²/f(x) = 1，则下列结论正确的个数是？
① f'(0) 存在且 =0；② f(0) 处取极小值；③ f''(0) 存在 = 0；④ f''(x) 在某邻域内连续
- **你的回答**：选了①②（f'(0)=0，且 f(0) 是极小值点），Q9③ 打了？表示不确定
- **缺漏**：Q9③ 不确定：由已知条件无法推出 f''(0) 存在。你在选项③ 后打？说明你意识到这一点，是对的直觉。
- **标准答案**：答案：B. 2个（①②）。
由 lim x²/f(x)=1 且 f 连续 → f(0)=0，f(x)~x²。
① f'(0)=lim f(x)/x = lim f(x)/x²·x = 1·0 = 0 ✓
② f(x)~x²，x→0 时 f(x)>0=f(0)，0 是极小值点 ✓
③ f''(0) 不一定存在 ✗（题目没给二阶可导条件）
④ 同理不一定连续 ✗

你打的？很准确——已知条件推不出 f''(0) 存在，所以③不能选。
- **错误**：否

## 高数研选题_第2章_Q11 · 数学二 · 含绝对值多项式的驻点与极值点判断 [fail · 2/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第11题：设 f(x)=|x²(x+1)| 的驻点个数为 m，极值点的个数为 n，则？
- **你的回答**：选 C（m=2,n=3），实际 m 和 n 都算错
- **缺漏**：驻点和极值点概念混淆。
【驻点（稳定点）定义详解】：
- 定义：f'(x₀)=0 的点 x₀ 称为驻点
- 注意：驻点 ≠ 极值点！f(x)=x³ 在 x=0 处 f'(0)=0 是驻点，但非极值点
- 驻点可能是：极值点（f' 变号）、非极值驻点（f' 不变号）

【极值点定义】：
- 若存在 x₀ 的某邻域，使得邻域内所有 x≠x₀ 都有 f(x)<f(x₀)（或 >），则 x₀ 是极值
- 判定：① f'(x₀)=0（必要条件）；② f' 在 x₀ 两侧变号（第一充分条件）
- **标准答案**：答案：D. m=3, n=2。
f(x)=|x²(x+1)|=x²|x+1|
绝对值零点：x=0（二重）、x=-1
各关键点分析：
- x=-2/3：f'=0 → 驻点 ✓，极大值 → 极值点 ✓
- x=-1：左右 f' 均 >0 → 驻点（f'(-1)=0）✓ 但不变号 → 非极值 ✗
- x=0：f'(0)=0 → 驻点 ✓，两侧≥0 → 极小值 → 极值点 ✓
驻点共 3 个(-2/3,-1,0)，极值点共 2 个(-2/3,0)
- **错误**：是

## 高数研选题_第2章_Q12 · 数学二 · 变限积分 + 极值点判断（莱布尼茨公式） [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第12题：设 f(x)=∫_x^{x²}(t-x sin t)^5 dt，求 f(x) 的极值点个数？
- **你的回答**：完全没思路，未作答
- **缺漏**：完全不会莱布尼茨公式处理含参变限积分。
- **标准答案**：解法：对 f(x) 求导（莱布尼茨公式）：
f'(x) = 2x·(x²-x sin(x²))^5 - 1·(x-x sin x)^5 - ∫_x^{x²} 5(t-x sin t)^4·(-sin t) dt

观察：第二项 = (x-x sin x)^5 = x^5(1-sin x)^5，在 x=0 处为 0
第一项在 x=0 处也为 0
积分项在 x=0 处也为 0

本题考研常见套路：直接对 f(x) 求导后观察 f'(x)=0 的点。
选择题技巧：x=0 时积分上下限相等 → f(0)=0；x=1 时积分上下限相等 → f(1)=0。

实际上本题需要进一步计算 f'(x) 后判断，核心考点：莱布尼茨公式 + 变限积分含参求导。
- **错误**：是

## 高数研选题_第2章_Q13 · 数学二 · 二阶导数极限 + 极值拐点联合判断 [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第13题：设 f 有二阶导数，lim_{x→0}[f(x)-a]/ln(1+x)=0，lim_{x→0}[f''(x)-1]/(e^{x²}-1)=2025，则？
A) f(0)是极大值 B) f(0)是极小值 C) (0,f(0))是拐点 D) f(0)不是极值也不是拐点
- **你的回答**：选A（误以为 f''(0)<0 是极大值），实际选B
- **缺漏**：由第二个极限推出 f''(0)=1>0，应是极小值，但误判为极大值。极值判据：f'(0)=0 且 f''(0)>0 → 极小值。
- **标准答案**：答案：B。
由第一个极限：ln(1+x)~x，所以 f(x)-a=o(x) → f(0)=a 且 f'(0)=0
由第二个极限：e^{x²}-1~x²，所以 [f''(x)-1]/(x²)→2025 → f''(0)=1>0
所以 f'(0)=0 且 f''(0)>0 → x=0 是极小值点 → 选 B。

你选了 A（极大值），根因：f''(0)>0 对应极小值，f''(0)<0 才对应极大值，搞反了！
- **错误**：是

## 高数研选题_第2章_Q14 · 数学二 · |f(x)| 的极值点与拐点判断 [fail · 4/10]

- **章节**：二、一元函数微分学
- **题目**：高数研选题 第2章 第14题：设 f 有二阶连续导数，f(0)=0, f'(0)>0, f''(0)<0，则关于 y=|f(x)| 在 x=0 处的说法？
- **你的回答**：选A（x=0 是 |f(x)| 的极值点，但 (0,f(0)) 不是拐点）
- **缺漏**：只判断了极值点部分正确，但拐点部分判断错了。

【拐点（inflection point）定义详解】：
- 定义：曲线 y=f(x) 凹凸性发生改变的点 (x₀, f(x₀))
- 注意：拐点是【曲线上的点】，不是 x 值本身
- 判定方法：① f''(x₀)=0 或 f''(x₀) 不存在；② f'' 在 x₀ 两侧【变号】
- 常见混淆：f''(x₀)=0 ⇏ 拐点（如 f(x)=x⁴，f''(0)=0 但不变号，不是拐点）

【|f(x)| 的拐点特殊判断】：
  令 g(x)=|f(x)|，看 g'' 符号：
  - x<0 侧：g(x)=-f(x) → g''(x)=-f''(x)
  - x>0 侧：g(x)=f(x) → g''(x)=f''(x)
  - 若 -f''(0⁻) 与 f''(0⁺) 异号 → 凹凸性改变 → 是拐点
- **标准答案**：答案：C（x=0 是 |f(x)| 的极值点，且 (0,f(0)) 是拐点）。
分析：
① f(0)=0, f'(0)>0 → 左侧 f<0 右侧 f>0
   所以 |f(x)|={-f(x), x≤0; f(x), x≥0}
② 极值：|f(0)|=0，两侧均正 → 极小值点 ✓
③ 令 g(x)=|f(x)|，看 g'' 符号：
   - x<0: g''=(-f)''=-f''，因 f''<0 连续 → g''>0 下凸
   - x>0: g''=f''<0 上凸
   凹凸性改变 → (0,0) 是拐点 ✓
结论：既是极值点也是拐点，选 C。
- **错误**：是
