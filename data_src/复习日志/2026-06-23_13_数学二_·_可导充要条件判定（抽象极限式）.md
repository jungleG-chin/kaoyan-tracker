# 数学二 · 可导充要条件判定（抽象极限式）

## K_可导充要条件判定 · 数学二 · 抽象极限式判定可导性 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：设 f(0)=0，则 f(x) 在点 x=0 可导的充要条件为哪个选项？
A) lim_{h→0} (1/h²)f(1-cosh) 存在
B) lim_{h→0} (1/h)f(1-e^h) 存在
C) lim_{h→0} (1/h²)f(h-sinh) 存在
D) lim_{h→0} (1/h)[f(2h)-f(h)] 存在
- **你的回答**：这是导数是否存在在某一点的判定，左推右和右推左的条件。
- **标准答案**：✓ 答案：B

【核心原理】
f'(0) 存在 ⇔ lim_{Δx→0} [f(0+Δx)-f(0)]/Δx = lim_{Δx→0} f(Δx)/Δx 存在。
关键判据：Δx 必须既能趋于 0+ 又能趋于 0-（双侧极限），且整个极限式必须严格凑成 f(Δx)/Δx 的形式。

【逐项分析】

① A 选项：lim (1/h²)f(1-cosh)
   当 h→0 时，1-cosh ≥ 0，只能趋于 0+（单侧）。
   所以 A 只能保证右导数存在，不能推出 f'(0) 存在。
   → 充分性不成立。

② B 选项：lim (1/h)f(1-e^h)
   令 t = 1-e^h，当 h→0 时 t→0，且 t 既能→0+ 也能→0-（双侧）。
   原式 = lim [f(t)/t] · [(1-e^h)/h] = f'(0) · (-1) = -f'(0)。
   极限存在 ⇔ f'(0) 存在。
   → 是充要条件，选 B。

③ C 选项：lim (1/h²)f(h-sinh)
   h-sinh ≈ h³/6，双侧趋于 0，前面的系数 1/h² 导致：
   原式 ≈ lim [f(h-sinh)/(h-sinh)] · [(h-sinh)/h²] ≈ f'(0) · 0 = 0。
   若 f'(0) 存在，极限为 0（必要性成立）；但极限存在不能反推 f'(0) 存在。
   → 只是必要条件，不是充分条件。

④ D 选项：lim (1/h)[f(2h)-f(h)]
   差分式不含 f(0) 项，无法保证 f 在 0 点连续，更无法保证可导。
   反例：f(x)=1 (x≠0), f(0)=0，则 [f(2h)-f(h)]/h = 0 极限存在，但 f 在 0 不连续，不可导。
   → 充分性不成立。

【左推右 vs 右推左】
• 左推右（可导 ⇒ 极限存在）：A、B、C、D 在 f'(0) 存在时极限都存在。
• 右推左（极限存在 ⇒ 可导）：只有 B 能反推。

【考试口诀】
「抽象极限判可导，先看增量是否双侧；再查能否凑定义，单侧/差分/系数归零都不行。」
- **错误**：否

## K_可导充要条件判定_总结 · 数学二 · 抽象极限式判定可导性：武忠祥总结框架 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：将极限式 lim_{h→0} f(φ(h))/ψ(h) 判定为 f'(0) 存在的充要条件，需要满足哪三个条件？
- **你的回答**：这是对上一道题的总结。
- **标准答案**：✓ 武忠祥老师总结框架：

要把 lim_{h→0} f(φ(h))/ψ(h) 作为 f(x) 在 x=0 可导的充要条件，必须同时满足以下三点：

① φ(h) → 0 且双侧（既能 →0⁺ 也能 →0⁻）
   • 保证能同时逼近左右导数
   • 反例 A：φ(h)=1-cosh 只能 →0⁺，只能推右导数

② φ(h) ≠ 0（当 h≠0 时）
   • 保证导数定义中的分母有意义
   • 否则会出现 0/0 之外的问题

③ φ(h) 与 ψ(h) 同阶
   • 即 lim_{h→0} φ(h)/ψ(h) = C（非零有限常数）
   • 这样 f(φ(h))/ψ(h) = [f(φ(h))/φ(h)] · [φ(h)/ψ(h)] 才能等价于 f'(0)·C
   • 反例 C：φ(h)=h-sinh ~ h³/6，ψ(h)=h²，二者不同阶，导致第二项趋于 0，无法反推 f'(0)

【用框架快速判题】

A) f(1-cosh)/h²：φ=1-cosh，ψ=h²
   ① 不满足（单侧）
   → 不是充要条件

B) f(1-e^h)/h：φ=1-e^h，ψ=h
   ① φ→0 双侧 ✓
   ② φ≠0 ✓
   ③ 1-e^h ~ -h，与 h 同阶 ✓
   → 是充要条件

C) f(h-sinh)/h²：φ=h-sinh，ψ=h²
   ① 双侧 ✓
   ② φ≠0 ✓
   ③ h-sinh ~ h³/6，与 h² 不同阶 ✗
   → 只是必要非充分

D) [f(2h)-f(h)]/h：
   ① 不是 f(φ(h)) 形式，而是差分式
   ② 不含 f(0)，不保证连续
   → 不是充要条件

【一句话总结】
抽象极限判可导 = 双侧 + 非零 + 同阶，缺一不可。
- **错误**：否

## K_绝对值可导与函数可导互推 · 数学二 · 绝对值函数可导性：f(x) 与 |f(x)| 的互推关系 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：f(x) 可导 与 |f(x)| 可导 之间的互推关系是什么？加上 f(x) 连续的前提后有何结论？
- **你的回答**：正常情况下两者没法互推；若 f(x) 连续，则 |f(x)| 可导能推出 f(x) 可导；进一步若 f(x₀)=0，还能推出 f'(x₀)=0。
- **标准答案**：✓ 武忠祥总结（【注】部分）：

【1. 一般情况：无法互推】
• f(x) 可导 ⇏ |f(x)| 可导
  反例：f(x)=x，|f(x)|=|x|，f 在 x=0 可导，但 |f| 不可导（尖点）
• |f(x)| 可导 ⇏ f(x) 可导
  反例：f(x) 在 x=0 不连续时，|f| 可能恰好可导，但 f 不可导

【2. 加上「f(x) 连续」前提后】
|f(x)| 在 x₀ 可导 → f(x) 在 x₀ 可导（右推左成立）

【3. 分两种子情况】

① 若 f(x₀) ≠ 0：
  |f(x)| 在 x₀ 处可导 ⟺ f(x) 在 x₀ 处可导（双向等价）
  理由：f(x₀)≠0 则 f 在 x₀ 附近保号，|f| = ±f，等价于±常数倍

② 若 f(x₀) = 0：
  |f(x)| 在 x₀ 处可导 ⟺ f'(x₀) = 0
  理由：f(x₀)=0 时曲线过 x 轴，|f| 会形成 V 形尖角；
  只有 f'(x₀)=0（切线水平）才能把尖角磨平，使 |f| 可导。
  注意：这个条件比「f 可导」更强——不仅要可导，导数值还必须是 0。

【速记口诀】
「无连续，不互推；加连续，右推左；
 函数值非零，双向等价；函数值为零，导数必须是零。」
- **错误**：否
