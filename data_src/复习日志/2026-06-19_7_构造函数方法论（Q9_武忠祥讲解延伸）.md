# 构造函数方法论（Q9 武忠祥讲解延伸）

## K_构造函数方法论 · 数学二 · 选择题构造反例函数的通用思维框架 [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：考研数学选择题中，如何想到构造一个特定函数来推翻某个选项？
- **你的回答**：不知道如何想到构造函数，感觉需要天赋或大量刷题
- **标准答案**：【核心结论】构造函数不是天赋，是一套可学习的反向工程。

【一、构造的本质】
题目给了约束条件 → 你在约束空间里找一个"坏函数"来推翻某选项。

【二、通用四步框架】
第1步：分析题目条件 → 确定函数的主项骨架
  - lim f(x)/x^n = c(非零有限) → 主项必须是 cx^n
  - f'(x0)=0 → 主项含 (x-x0)^2
  - f(x0)=0, f'(x0)!=0 → 主项是 k(x-x0)

第2步：明确要推翻哪个选项 → 决定扰动类型
  - 推翻极值 → 需要函数变号 → 加振荡项
  - 推翻连续性/可导性 → 加尖角或振荡
  - 推翻导数连续性 → 加高阶振荡

第3步：设计扰动项 → 强度必须比主项低阶
  - 振幅 O(x^{n+1}) 不影响主项极限 ✓
  - 振幅 O(x^n) 或更高会破坏已知条件 ✗

第4步：组合验证 → 主项+扰动项是否满足所有条件

【三、四个万能零件】
零件A: x^n * sin(1/x) —— 振荡器（破坏极值/连续性）
零件B: (x-x0)^n —— 平滑器（保证某阶导数为零）
零件C: |x-x0| —— 尖角器（制造不可导点）
零件D: 分段定义 —— 拼接器（左右行为不同时）

【四、以 Q9 为例演示】
条件: lim(x->0) x^2/f(x)=1 → f(x)~x^2（主项=x^2）
要推翻选项2(极小值): 需要 f(x) 在 0 附近变号
→ 加扰动 x^3*sin(1/x)（振幅 O(x^3)，比 x^2 低一阶）
→ f(x)=x^2+x^3*sin(1/x)
→ 主项保住极限=1 ✓，扰动让函数变号 → 不是极值 ✗
→ 同理选项4(f''连续)也被这个反例推翻

【五、考场保底策略（不会构造也能做）】
选择题不需要写出具体构造！只需判断"能不能确定"：
f(x)~x^2 只限制了主项，细节未知
→ f(x)可能=x^2（此时②③④全对）
→ f(x)也可能=x^2+x^3sin(1/x)（此时②③④全错）
→ 条件不够 → ②③④都不能确定
→ 只有①能从定义严格推出
→ 选 A

【六、如何训练这种能力】
1. 积累零件库：10个常用构造背下来就行
2. 训练反向思维：每学一个定理就想"逆命题为什么不对"
3. 精做题：每道错题按框架拆解，不追求题量追求深度
