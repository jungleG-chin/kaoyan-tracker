# 9 题弱项分析（数学极限章节）

## Q18 · 数学二 · 分段函数 + 间断点 [fail · 3/10]

- **章节**：一、函数、极限、连续
- **题目**：间断点 4 类判据
- **你的回答**：不会系统分类
- **缺漏**：不会系统分类 4 类间断点
- **标准答案**：✓ 间断点 4 类判据：
① 可去：左右极限存在且相等。
② 跳跃：左右极限存在但不等。
③ 无穷：左右极限至少一个为 ±∞。
④ 振荡：极限不存在且不趋于无穷。
①② 统称第一类，③④ 统称第二类。
- **错误**：是

## Q23 · 数学二 · 1^∞ 型 + 泰勒联用 [fail · 4/10]

- **章节**：一、函数、极限、连续
- **题目**：1^∞ 型极限 + 泰勒展开
- **你的回答**：写了 sin 展开但漏 ln y = g·(f-1)
- **缺漏**：卡在 g·(f-1) 这一步
- **标准答案**：1^∞ 三步法：
① 判型：f→1, g→∞
② 取对数：ln y = g·ln f
③ 等价：ln(1+u)~u，所以 ln y ~ g·(f-1)
④ 还原 e^[lim g·(f-1)]
配合泰勒：先把 f 展到 g·(f-1) 不再含 ∞ 的阶。
- **错误**：是

## Q25 · 数学二 · 夹逼放缩过宽 [fail · 3/10]

- **章节**：一、函数、极限、连续
- **题目**：夹逼放缩过宽导致无法夹住
- **你的回答**：夹逼放缩过宽，右侧比真实量大很多
- **缺漏**：放缩的'度'控制不住
- **标准答案**：夹逼放缩的'度'：
- 放上界：分母中取最大 → 分母变小 → 整体放大
- 放下界：分母中取最小 → 分母变大 → 整体缩小
- 关键：放大缩小后极限相同才能夹住。
- **错误**：是

## Q27 · 数学二 · 分式泰勒展开 + 待定系数 [fail · 4/10]

- **章节**：一、函数、极限、连续
- **题目**：泰勒展开 (1+ax)/(1+bx) 并待定系数
- **你的回答**：(1+ax)/(1+bx) 展错
- **缺漏**：1/(1+bx) 几何级数展错
- **标准答案**：(1+ax)/(1+bx) = (1+ax)·(1-bx+b²x²-b³x³+...)
= 1 + (a-b)x + (b²-ab)x² + ...
配合待定系数法：哪一项系数为 0 列方程。
- **错误**：是

## Q32 · 数学二 · x^x 型极限 + 差 [fail · 2/10]

- **章节**：
- **题目**：lim x→0⁺ [x^x - (sin x)^x] / [x² ln(1+x)]
- **你的回答**：写不下去，不知道拆主导项
- **缺漏**：差的主导项识别
- **标准答案**：x^x = e^(x ln x) ≈ 1 + x ln x
(sin x)^x ≈ 1 + x ln(sin x)
差 = x·[ln x - ln(sin x)] ≈ x·(x - x³/6) = x² - x⁴/6
分母 = x²·(x - x²/2 + ...) ≈ x³
所以原式 ≈ (x²)/x³ = 1/x → ∞
- **错误**：是

## Q34 · 数学二 · 积分上限函数 [fail · 0/10]

- **章节**：
- **题目**：lim x→+∞ ∫₁ˣ [t²(e^(1/t)-1) - t]dt / [x² ln(1+1/x)]
- **你的回答**：未动笔
- **缺漏**：积分上限函数求导 + 变上限代换
- **标准答案**：分子：t²(e^(1/t)-1) - t ≈ t²·(1/t) - t = 0（等价无穷小）
分母：x²·(1/x) = x
整体：0/x → 0
关键：先把积分里的被积函数做等价无穷小化简。
- **错误**：是

## Q(2) · 数学二 · 嵌套极限 1^∞ 变体 [fail · 2/10]

- **章节**：
- **题目**：lim x→+∞ (x^(1/x) - 1)^(1/ln x)
- **你的回答**：尝试 e^x 代换，方向跑偏
- **缺漏**：未识别 1^∞ 型
- **标准答案**：底 A = x^(1/x) - 1 → 1⁺，指数 B = 1/ln x → 0
是 (1+·)^0 型 → 用 1^∞ 框架：
ln y = (1/ln x)·ln(x^(1/x) - 1)
≈ (1/ln x)·(x^(1/x) - 1)
≈ (1/ln x)·(e^(ln x / x) - 1)
≈ (1/ln x)·(ln x / x)
= 1/x → 0
所以原式 = e^0 = 1。
- **错误**：是

## Q(4) · 数学二 · 1^∞ 型 (tan^n) [fail · 0/10]

- **章节**：
- **题目**：lim n→∞ tan^n(π/4 + 2/n)
- **你的回答**：未动笔
- **缺漏**：1^∞ 标准处理
- **标准答案**：tan(π/4 + 2/n) → 1，n→∞，是 1^∞
ln y = n·ln tan(π/4 + 2/n)
≈ n·(tan(π/4 + 2/n) - 1)
≈ n·(sec²(π/4)·2/n)
= n·(2·2/n) = 4
所以原式 = e^4。
- **错误**：是
