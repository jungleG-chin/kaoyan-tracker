# 数学二 · 严选题第二章 · 15-23题

## Q15 · 数学二 · 拐点判定（二/三阶导判断） [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：设f(x)满足f'(0)=0, f'(x)+[f(x)]^3=x^2，判断(0,f(0))是否为拐点。
- **你的回答**：二阶导在该点为零，三阶导>0，则左边二阶导<0，右边二阶导>0，左凸右凹，(0,f(0))是拐点，选(C)。
- **标准答案**：正确。关键步骤：
1. 对方程两边求导得 f''(x)+3[f(x)]^2·f'(x)=2x
2. 代入x=0: f''(0)+0=0 → f''(0)=0（嫌疑点）
3. 再求导得 f'''(x)+3[2f·(f')^2+f^2·f'']=2
4. 代入x=0: f'''(0)=2>0
5. 由 f''(0)=0, f'''(0)>0 → f''在x=0处从负变正（拐点）
结论：(0,f(0))是曲线 y=f(x) 的拐点，选(C)。
- **错误**：否

## Q16 · 数学二 · 渐近线条数（偶函数斜渐近线） [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：曲线 y=(x^2+1)/sqrt(x^2-1) 的渐近线条数为？
- **你的回答**：选(A)1。（错误，未考虑偶函数两端都有斜渐近线）
- **缺漏**：偶函数 y=f(x) 当 x→+∞ 和 x→-∞ 若各有一条斜渐近线，由于偶函数对称性，两侧斜渐近线斜率互为相反数，即共2条不同的斜渐近线。另外还有竖直渐近线 x=±1，所以共4条。
- **标准答案**：答案(D)4条。分析：
1. 竖直渐近线：x^2-1=0 → x=±1，2条
2. 斜渐近线：y是偶函数，且x→∞时可求斜率a=lim y/x=1
   x→+∞时 y=x+0，x→-∞时 y=-x+0（偶函数特性，两边斜率±1）
   共2条斜渐近线
3. 总计：2+2=4条
⚠️ 易错点：见到偶函数求渐近线，先想到两端斜渐近线可能不一样！
- **错误**：是

- **题目**：偶函数的斜渐近线条数规律是什么？
- **你的回答**：
- **标准答案**：若 f(x) 是偶函数，且 x→+∞ 时有斜渐近线 y=ax+b，则 x→-∞ 时斜渐近线为 y=-ax+b（斜率符号相反），因此偶函数有斜渐近线时一定是2条（不是1条）。

## Q18 · 数学二 · 极限计算（导数定义 + 特殊函数构造） [warn · 0/10]

- **章节**：二、一元函数微分学
- **题目**：设曲线 y=f(x) 与 y=x^2-x 在点(1,0)处有公共切线，求 lim_{n→∞} n·f(n/(n+2))。
- **你的回答**：答案-2，但做题过程写错。
- **缺漏**：两个关键方法没想到：
① 已知函数值+导数值 → 凑导数定义
② 构造特殊函数（满足题目抽象条件的具体函数来验证）
- **标准答案**：解题步骤：
1. 由公共切线条件：f(1)=0, f'(1)=2·1-1=1
2. lim n·f(n/(n+2)) = lim n·f(1 - 2/(n+2))
3. 令 t=n/(n+2)，t→1时 n=2t/(1-t)
4. 换元后 = lim [f(t)-f(1)]/(t-1) · lim n(t-1) = f'(1)·(-2) = 1·(-2) = -2
⚠️ 方法1：题目给了某点函数值+导数值 → 大概率要凑导数定义！
⚠️ 方法2：构造满足条件的具体函数（如令f(x)=x-1）验证答案。
- **错误**：是

## Q19 · 数学二 · 导数定义（积分型函数求导） [warn · 0/10]

- **章节**：二、一元函数微分学
- **题目**：已知 f(x)=[(x-1)(x-2)...(x-n)]/[(x+1)(x+2)...(x+n)]，求 f'(1)。
- **你的回答**：用极限定义 f'(1)=lim[f(x)-f(1)]/(x-1)，当时没想到直接代。
- **缺漏**：f(1)=0（分子含(x-1)因子，x=1时为0）。利用导数定义：f'(1)=lim_{x→1}[f(x)-0]/(x-1)=lim_{x→1}[(x-2)...(x-n)]/[(x+1)...(x+n)]，直接代x=1即可。
- **标准答案**：f'(1) = [(1-2)(1-3)...(1-n)] / [(1+1)(1+2)...(1+n)]
= [(-1)(-2)...(-n+1)] / [2·3...·(n+1)]
= (-1)^{n-1}·(n-1)! / (n+1)!/1
= (-1)^{n-1}·(n-1)! / (n+1)!
⚠️ 关键：看到乘积型函数求某点导数，优先想导数定义，因为f(1)=0可以直接约掉(x-1)。
- **错误**：是

## Q20 · 数学二 · 参数方程求切线 [pass · 10/10]

- **章节**：二、一元函数微分学
- **题目**：参数方程曲线 x=∫₀^t e^{-u²}du, y=t²ln(2-t²) 在点(0,0)处的切线方程。
- **你的回答**：方法：求切线先求对应点的t值，再得到dy/dx和dt的关系。
- **标准答案**：1. 由(0,0)知 x=0 → ∫₀^t e^{-u²}du=0 → t=0
2. dy/dx = (dy/dt)/(dx/dt)
   dx/dt = e^{-t²}
   dy/dt = 2t·ln(2-t²) + t²·(-2t)/(2-t²)
   t=0时：dx/dt=1, dy/dt=0
3. 切线斜率=0/1=0，切线方程 y=0
⚠️ 记忆点：参数方程求切线 → 先找t值，再用链式法则 dy/dx=(dy/dt)/(dx/dt)。
- **错误**：否

## Q22 · 数学二 · 复合函数求导（链式法则） [warn · 0/10]

- **章节**：二、一元函数微分学
- **题目**：设 f(x)={ln√x (x≥1), 2x-1 (x<1)，y=f(f(x))，求 dy/dx|_{x=e}。
- **你的回答**：先分段讨论了各分段，做复杂了，其实直接用链式求导法则然后代入x=e就行。
- **缺漏**：链式法则：y'=[f(f(x))]' = f'(f(x))·f'(x)。代入x=e：f(e)=ln√e=1/2<1，所以用第二段求f'(f(x))；f'(e)用第一段求。不需要分段讨论整个y，直接代就行。
- **标准答案**：y'=f'(f(x))·f'(x)
x=e时：
  f(e)=ln√e = 1/2 < 1
  f'(x)=1/(2x)（x≥1），f'(e)=1/(2e)
  f'(f(e))=f'(1/2)=2（用第二段 f'(x)=2，x<1）
  dy/dx|_{x=e} = 2 · 1/(2e) = 1/e
⚠️ 错误：不要把复合函数 y=f(f(x)) 展开分段讨论，直接链式法则代值即可。
- **错误**：是

## Q23 · 数学二 · 反函数二阶导 [fail · 0/10]

- **章节**：二、一元函数微分学
- **题目**：设 y=f(x) 的反函数是 x=φ(y)，且 f(x)=∫₁^{2x} e^{t²}dt+1，求 φ''(1)。
- **你的回答**：这个点掌握不太行。
- **缺漏**：反函数求导公式没有熟练掌握。
- **标准答案**：反函数求导公式：
  φ'(y) = 1/f'(x)
  φ''(y) = -f''(x)/[f'(x)]^3

步骤：
1. f(x)=∫₁^{2x}e^{t²}dt+1，f'(x)=e^{(2x)²}·2=2e^{4x²}
2. f''(x)=2·16x·e^{4x²}=32x·e^{4x²}
3. 求当φ(1)=?即 f(x)=1 → ∫₁^{2x}e^{t²}dt=0 → 2x=1 → x=1/2
4. f'(1/2)=2e^1=2e；f''(1/2)=32·(1/2)·e=16e
5. φ''(1) = -f''(1/2)/[f'(1/2)]^3 = -16e/(2e)^3 = -16e/(8e^3) = -2/e^2
⚠️ 关键公式：φ''(y) = -f''(x) / [f'(x)]^3（须先找对应的x值）。
- **错误**：是
